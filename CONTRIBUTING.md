# Contributing

## File Structure

### rules

Contains a set of language/framework based rules to be followed along with 2 special folders **doc-common** and **undoc-common**. **doc-common** will contain the common rules' file(s) for programming languages who allow some form of commenting is allowed (like *Javascript* allows `//`, `/**/` and `/** */`) whereas **undoc-commong will contain the common rules' file(s) for the programming languages who doesn't allow any form of commenting (like *Brainf\*ck*).
